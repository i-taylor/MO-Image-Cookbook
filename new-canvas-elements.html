    Here are the call-out box UI options:

    At this time, the color combinations are approved for use only on the white background of text-block sections.

    Information

    Use this type of call-out box for informational instances, such as "definitions" and "key takeaways."

    To use the informational call-out box, copy/paste/edit the following code:

    <div class="callout-box">
      <p class="tag">Tag label</p>
      <p>Put content here</p>
    </div>

    action

    Use this type of call-out box for instances where you want the learner to take an action, such as "reflections."

    To use the action call-out box, copy/paste/edit the following code:

    <div class="callout-box action">
      <p class="tag">Tag label</p>
      <p>Put content here</p>
    </div>

    highlight

    This type of call-out box is for future use, anticipating situations where we need to warn learners or otherwise be brightly colorful.

    To use the bright blue call-out box, copy/paste/edit the following code:

    <div class="callout-box highlight">
      <p class="tag">Tag label</p>
      <p>Put content here</p>
    </div>
