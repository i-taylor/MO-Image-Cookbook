Michigan Online — Snippet Rules

Guidance for consistent, accessible image snippets.

Standard snippet rules decorative images
  • All snippets use: role="presentation", alt="", and include explicit width/height, with no trailing <br />.
  • Base URL: https://shared-files.online.umich.edu/Michigan%20Online/Deployed/
  • Course Images: https://shared-files.online.umich.edu/Michigan%20Online/Deployed/Course%20Images/
  • Icons: https://shared-files.online.umich.edu/Michigan%20Online/Deployed/Icons/
Image Assets Snippet Rules

Snippet checklist

Use these every time you add a new image.

  1. Use role="presentation" and alt="" for decorative assets.
  2. Include explicit width and height attributes.
  3. Use the correct base URL for the image category.
  4. Avoid trailing <br /> tags in snippets.
<img role="presentation" src="https://shared-files.online.umich.edu/Michigan%20Online/Deployed/Icons/video.png" alt="" width="84" height="84" />

Accessibility note: These assets are treated as decorative. If an image conveys meaning, remove role="presentation" and replace alt="" with meaningful alt text.
